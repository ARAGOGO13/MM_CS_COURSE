# Семестр 2 — Задание 8. Прямые методы решения линейных систем

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки: `n` – размер матрицы `A`, `p` – количество выводимых значений, `k` – номер формулы инициализации (0 = файл), `filename`. Например, `./a.out 4 4 0 a.txt` или `./a.out 2000 6 1`.
2. В задачах, где требуется правая часть `b`, она строится после инициализации матрицы $A=(a_{i,j})$ по формуле $b_i = \sum_{k=0}^{(n-1)/2} a_{i,2k+1}$, отдельной подпрограммой, вызываемой из `main`.
3. Ввод матрицы должен быть оформлен в виде подпрограммы, находящейся в отдельном файле.
4. Ввод матрицы из файла: `n × n` чисел построчно; программа выводит сообщение об ошибке при проблемах чтения.
5. Ввод матрицы и правой части по формуле $a_{i,j} = f(k, n, i, j)$ — та же таблица, что в [Задании 4](../Tasks04/TASK.md) (с `n` вместо `n,m`).
6. Решение должно быть оформлено в виде функции в отдельном файле, получающей: размерность `n`, матрицу `A`, правую часть `b` (для задач решения системы), вектор `x` для решения или матрицу `X` для обратной матрицы, и дополнительные векторы, если требуется. Получать доп. информацию через глобальные переменные запрещается.
7. Функция возвращает ненулевое значение, если алгоритм неприменим к данной матрице `A`.
8. Функция не должна выделять или использовать дополнительную память.
9. Сложность работы функции не должна превышать $O(n^3)$.
10. Суммарный объём памяти: не более $n^2 + O(n)$ при решении системы, $2n^2 + O(n)$ при вычислении обратной матрицы. Для соблюдения этого после завершения алгоритма вызывается подпрограмма повторной инициализации матрицы/`b`.
11. Программа должна содержать подпрограмму вывода `n × n` матрицы, не более `p` строк/столбцов, формат элемента `" %10.3e"`.
12. Результаты — 3 элемента: вектор решения `x` (или обратная матрица $A^{-1}$) и два числа $r_1$, $r_2$ (подпрограмма без выделения памяти):
    - Для решения системы: $r_1 = \|Ax-b\|_1/\|b\|_1$, $r_2 = \left(\sum_{i=1}^n |x_i - (i \bmod 2)|\right)/\sum_{i=1}^n (i\bmod2)$, где $\|y\|_1 = \sum_i |y_i|$.
    - Для обратной матрицы (при $n \le 4000$, иначе 0): $r_1 = \|AA^{-1}-E\|_1$, $r_2 = \|A^{-1}A-E\|_1$, где $\|Y\|_1 = \max_{1\le j\le n} \sum_{i=1}^n |y_{ij}|$.
13. Вывод результата в `main`: вектор `x` или матрица $A^{-1}$ (через подпрограмму печати), затем

    ```c
    printf ("%s : Task = %d Res1 = %e Res2 = %e Elapsed = %.2f K = %d N = %d\n",
            argv[0], task, r1, r2, t, k, n);
    ```
