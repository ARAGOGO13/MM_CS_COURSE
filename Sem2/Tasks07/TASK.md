# Семестр 2 — Задание 7. Итерационные методы решения линейных систем

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Во всех задачах требуется вычислить вектор — `m`-й элемент указанной в условии последовательности, строящейся по заданной `n×n` матрице `A` и 0-му элементу последовательности `x0`.
3. Аргументы командной строки для задач 1, 3–6: `m` – номер элемента последовательности, `n` – размер матрицы `A`, `p` – количество выводимых значений, `kA`/`fA` (матрица `A`), `kx`/`fx` (вектор `x0` размера `n×1`). Например, `./a.out 100 6 5 0 a.txt 0 x.txt` или `./a.out 100 2000 6 1 4`.
4. Аргументы командной строки для задач 2, 7–10: то же самое плюс дополнительный первый аргумент `τ`.
5. Ввод матрицы должен быть оформлен в виде подпрограммы, находящейся в отдельном файле.
6. Ввод матрицы из файла: `n × n` чисел построчно; программа выводит сообщение об ошибке при проблемах чтения.
7. Ввод матрицы и правой части по формуле $a_{i,j} = f(k, n, i, j)$ — та же таблица, что в [Задании 4](../Tasks04/TASK.md).
8. В задачах 2–10, где участвует вектор `b`, он строится после инициализации матрицы $A = (a_{i,j})$ по формуле $b = (b_i)_{i=1,\ldots,n}$, $b_i = \sum_{k=0}^{(n-1)/2} a_{i, 2k+1}$. Инициализация оформляется отдельной подпрограммой, вызываемой из `main`.
9. Решение должно быть оформлено в виде подпрограммы, находящейся в отдельном файле.
10. Программа должна содержать подпрограмму вывода на экран `m × n` матрицы, не более `p` строк и столбцов, формат элемента `" %10.3e"`.
11. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
12. Сложность работы подпрограммы не должна превышать $C(m+1) n^2$ при $n, m \to \infty$: $C=1$ в задачах 1–7, $C=3/2$ в задачах 8, 9, $C=2$ в задаче 10. При переходе от $x_{k-1}$ к $x_k$ допускается только одно умножение матрицы `A` на вектор; в задачах 8–10 дополнительно решается система с треугольной матрицей методом последовательного исключения неизвестных.
13. Результаты работы задач 1–10 — три элемента: вектор $x_m$ и два числа $r_1$, $r_2$, вычисляемых отдельной подпрограммой (без выделения памяти), вызываемой из `main`:
    - Для задачи 1: $r_1$ — возвращаемое значение функции, $r_2 = \left(\sum_{i=1}^n \left(\sum_{j=1}^n a_{ij}x_j - r_1 x_i\right)\right) / \sum_{i=1}^n |r_1 x_i|$.
    - Для задач 2–10: $r_1 = \left(\sum_{i=1}^n \left(\sum_{j=1}^n a_{ij}x_j - b_i\right)\right) / \sum_{i=1}^n |b_i|$, $r_2 = \left(\sum_{i=1}^n |x_i - (i \bmod 2)|\right) / \sum_{i=1}^n (i \bmod 2)$, где $(x_i)$ — компоненты вектора $x_m$.
14. Вывод результата в `main`: вектор `x_m` (вызовом подпрограммы печати матрицы размера `1×n`), затем

    ```c
    printf ("%s : Task = %d Res1 = %e Res2 = %e Elapsed = %.2f\n",
            argv[0], task, r1, r2, t);
    ```

## Задачи

Во всех задачах: функция/подпрограмма получает `n×n` матрицу `A`, вектора `x0`, (`b`, `r`, `w` — где нужны), числа `n`, `m` (и `τ`, где нужно), и возвращает в векторе `x` значение $m$-го члена соответствующей последовательности $\{x_k\}$; $x_0$ — 0-й элемент, значение которого может быть изменено подпрограммой; $(\cdot,\cdot)$ — евклидово скалярное произведение.

1. $\lambda_k = (Ax_k, x_k)/(x_k, x_k)$, $x_k = Ax_{k-1}$ — метод отношения Рэлея (степенной метод). Возвращает $m$-й член $\{\lambda_k\}$.
2. $(x_k - x_{k-1})/\tau + Ax_{k-1} = b$ — простая итерация с постоянным шагом `τ`.
3. $(x_k - x_{k-1})/\tau_{k-1} + Ax_{k-1} = b$, $\tau_k = (r_k, r_k)/(Ar_k, r_k)$, $r_k = Ax_k - b$ — метод скорейшего спуска (`r` — доп. память).
4. $(x_k - x_{k-1})/\tau_{k-1} + Ax_{k-1} = b$, $\tau_k = (Ar_k, r_k)/(Ar_k, Ar_k)$, $r_k = Ax_k - b$ — метод минимальных невязок.
5. $D(x_k - x_{k-1})/\tau_{k-1} + Ax_{k-1} = b$, $\tau_k = (D^{-1}r_k, r_k)/(AD^{-1}r_k, D^{-1}r_k)$, $D$ — диагональ `A` — метод скорейшего спуска с диагональным предобуславливанием.
6. То же с $\tau_k = (AD^{-1}r_k, r_k)/(AD^{-1}r_k, AD^{-1}r_k)$ — метод минимальных невязок с диагональным предобуславливанием.
7. $D(x_k - x_{k-1})/\tau + Ax_{k-1} = b$, `τ` — постоянный шаг, `D` — диагональ `A`.
8. $(D+L)(x_k - x_{k-1})/\tau + Ax_{k-1} = b$, `D` — диагональ, `L` — нижняя треугольная часть `A` — нижний метод Зейделя.
9. $(D+R)(x_k - x_{k-1})/\tau + Ax_{k-1} = b$, `R` — верхняя треугольная часть `A` — верхний метод Зейделя.
10. $(D+L)D^{-1}(D+R)(x_k - x_{k-1})/\tau + Ax_{k-1} = b$ — симметричный метод Зейделя (SSOR).
