# Семестр 2 — Задание 9. Интерполяция и ряды Тейлора

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки для задач 1–3: `x0` – точка вычисления, `n` – число узлов интерполяции, `f` – имя файла с массивами `x`, `y` (не менее `2n` чисел, построчно `xi yi`, где $y_i = f(x_i)$). Например, `./a01.out 1e3 6 a.txt`.
3. Аргументы командной строки для задачи 4: `x0`, `n`, `f` – имя файла с массивами `x`, `y`, `d` (не менее `3n` чисел, построчно `xi yi di`, где $d_i = f'(x_i)$).
4. Аргументы командной строки для задач 5–8: `x` – точка вычисления, `ε` – точность. Например, `./a05.out 1e3 1e-14`.
5. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
6. Вывод результата в `main`:
   - Для задач 1–4: `printf ("%s : Task = %d Result = %e Elapsed = %.2f\n", argv[0], task, r, t);`
   - Для задач 5–8: `printf ("%s : Task = %d Result = %e Residual = %e Elapsed = %.2f\n", argv[0], task, r1, r2, t);`, где `r2` — модуль разности между вычисленным и истинным (библиотечным) значением функции.

## Задачи

1. Значение интерполяционного многочлена Лагранжа по узлам `x[n]`, `y[n]` в точке `x0`, построенного по **определению**.
2. То же самое, но по **интерполяционной формуле Ньютона** с разделёнными разностями.
3. То же самое, но по **схеме Эйткена**.
4. Значение в точке `x0` интерполяционного многочлена Лагранжа по узлам `x[n]`, `y[n]` с заданными производными `d[n]` (интерполяционная формула Ньютона с кратными узлами).
5. Значение $\sin(x)$ в точке `x` с точностью `ε`, вычисленное суммированием ряда Тейлора.
6. То же самое для $\cos(x)$.
7. То же самое для $\exp(x)$.
8. То же самое для $\ln(x)$.
