# Семестр 2 — Задание 2. Массив строк: фильтрация

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки: `n` – длина массива строк, `p` – максимальное количество выводимых строк, `fin` – имя входного файла со строками, `s` – строка. Например, `./a01.out 10 5 a.txt "abcd"`.
3. Ввод массива строк должен быть оформлен в виде функции. Строки текстового файла (без символа конца строки) считываются как элементы массива строк. Функция должна возвращать признак ошибки, если файл не может быть прочитан, содержит меньше данных, данные неверного формата или недостаточно памяти.
4. Решение задачи должно быть оформлено в виде функции, получающей массив, его длину и строку `s`. Получать дополнительную информацию извне и выводить что-либо на экран в этой функции запрещается.
5. Программа должна содержать подпрограмму `print_array` вывода на экран массива длины не более `p`, по одному элементу на строку.
6. Программа должна содержать подпрограмму `free_array`, получающую массив, его длину, и удаляющую всю выделенную под массив строк память.
7. Результатом работы каждой функции является измененный массив и его новая длина (оба выводятся в `main`). Никакие функции не должны обращаться к элементам за новым концом массива.
8. Вывод результата работы функции в функции `main`:

   ```c
   printf ("New array:\n");
   print_array (a, res, p); /* вывод массива a длины res */
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   free_array (a, res); /* удаление массива a длины res */
   ```

## Задачи

Во всех задачах: функция получает массив строк `a`, целое число `n` (длину массива) и строку `s`, выбрасывает элементы согласно условию (не более $n + O(1)$ пересылок) и возвращает новую длину массива.

1. Выбросить все элементы, лексикографически меньшие строки `s`.
2. Выбросить все элементы, которые входят в строку `s`.
3. Выбросить все элементы, не имеющие общих символов со строкой `s`.
4. Выбросить все подряд идущие элементы, имеющие длину меньше, чем строка `s`, оставляя только первый элемент в таком участке.
5. Выбросить все подряд идущие элементы, у которых конец строки совпадает со строкой `s`, оставляя только первый элемент в таком участке.
6. Выбросить все подряд идущие элементы, которые не имеют общих символов со строкой `s`, оставляя только первый и последний элементы в таких участках.
7. Выбросить все элементы, имеющие соседа (слева или справа), лексикографически меньшего строки `s`.
8. Выбросить все элементы, имеющие соседа (слева или справа), который входит в строку `s` (прописные и соответствующие строчные латинские буквы считаются совпадающими).
