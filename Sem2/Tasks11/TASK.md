# Семестр 2 — Задание 11. Численное дифференцирование и интегрирование

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Задачи оцениваются независимо в двух группах: задачи 1–3 и задачи 4–12.
3. Аргументы командной строки для задач 1–3: `x` – точка, `h` – параметр, `k` – номер функции. Например, `./a01.out 1 1e-8 2`.
4. Аргументы командной строки для задач 4–7: `a`, `b` – концы отрезка, `n` – число точек, `k` – номер функции.
5. Аргументы командной строки для задач 8, 9: `a`, `b`, `ε` – параметр, `k`.
6. Аргументы командной строки для задач 10, 11: `a`, `ε`, `k`.
7. Аргументы командной строки для задачи 12: `a`, `b`, `ε`, `kx` – номер функции `x`, `ky` – номер функции `y`.
8. В программе должны быть реализованы подпрограммы для функций $f(x)$ в зависимости от `k`: $f(x) = 1$ ($k{=}0$), $1+x$ ($k{=}1$), $1+x+x^2$ ($k{=}2$), $1+x+x^2+x^3$ ($k{=}3$), $1+x+x^2+x^3+x^4$ ($k{=}4$), $e^{-x}$ ($k{=}5$), $1/(25x^2+1)$ ($k{=}6$).
9. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
10. Вывод результата в `main`:
    - Для задач 1–7: `printf ("%s : Task = %d Res = %e Count = %d T = %.2f\n", argv[0], task, res, count, t);`
    - Для задач 8, 9, 12: `printf ("%s : Task = %d Res = %e N = %d Count = %d T = %.2f\n", argv[0], task, res, n, count, t);`
    - Для задач 10, 11: `printf ("%s : Task = %d Res = %e B = %e Count = %d T = %.2f\n", argv[0], task, res, b, count, t);`

    где `count` – число вызовов функции `f`, `n`/`b` – возвращаемое значение функции (число итераций/конечный верхний предел).

## Задачи

1. Приближённое значение $f'(x)$ с погрешностью $O(h)$: $f'(x) = (f(x+h) - f(x))/h$.
2. Приближённое значение $f'(x)$ с погрешностью $O(h^2)$: $f'(x) = (f(x+h) - f(x-h))/(2h)$.
3. Приближённое значение $f''(x)$ с погрешностью $O(h^2)$: $f''(x) = (f(x+h) - 2f(x) + f(x-h))/h^2$.
4. Приближённое значение $\int_a^b f(x)\,dx$ по составной формуле трапеций с погрешностью $O(h^2)$: $h(f(a)/2 + f(a+h) + \cdots + f(a+(n-1)h) + f(b)/2)$, $h=(b-a)/n$.
5. Приближённое значение $\int_a^b f(x)\,dx$ по составной формуле Симпсона с погрешностью $O(h^3)$: $(2/3)h(f(a)/2 + 2f(a+h) + f(a+2h) + \cdots + 2f(a+(2n-1)h) + f(b)/2)$, $h=(b-a)/(2n)$.
6. Приближённое значение $\int_a^b \dfrac{f(x)}{\sqrt{|x|}}\,dx$ по составной формуле трапеций с весом $1/\sqrt{|x|}$.
7. То же самое по составной формуле Симпсона с весом $1/\sqrt{|x|}$.
8. Значение $\int_a^b f(x)\,dx$ с точностью `ε` по составной формуле трапеций с автоматическим выбором шага. Возвращает конечное `n` (или отрицательное значение при неудаче).
9. То же самое по составной формуле Симпсона.
10. Значение несобственного интеграла $\int_a^{+\infty} f(x)\,dx = \lim_{b\to\infty}\int_a^b f(x)\,dx$ с точностью `ε` по составной формуле трапеций с автоматическим выбором шага и предела `b`. Возвращает конечное значение `b`.
11. То же самое по составной формуле Симпсона.
12. Длина кривой $(x(t), y(t))$ при `t` от `a` до `b` с точностью `ε`, вычисляемая как предел сумм длин ломаных с вершинами $(x(a),y(a)), (x(a{+}h),y(a{+}h)), \ldots, (x(b),y(b))$ при $n\to\infty$, $h=(b-a)/n$. Возвращает конечное значение `n`.
