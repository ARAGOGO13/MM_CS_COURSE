# Семестр 2 — Задание 4. Матрицы: базовые операции

## Требования к программам

1. Задачи оцениваются независимо в двух группах: задачи 1–5 и задачи 6–10.
2. Программа должна получать все параметры в качестве аргументов командной строки.
3. Аргументы командной строки для задач 1–5: `n` – размерность матрицы, `p` – количество выводимых значений, `k` – номер формулы инициализации (0 = чтение из файла), `filename` – имя файла (отсутствует, если `k != 0`). Например, `./a.out 4 4 0 a.txt` читает матрицу 4×4 из файла; `./a.out 2000 6 1` инициализирует матрицу 2000×2000 по формуле 1.
4. Аргументы командной строки для задач 6–7: `i`, `j` – параметры задачи, `n` – число строк, `m` – число столбцов, `p`, `k`, `filename` — как выше. Например, `./a.out 1 2 6 4 4 0 a.txt`.
5. Аргументы командной строки для задачи 8: `i`, `j`, `γ` (вещественный параметр), `n`, `m`, `p`, `k`, `filename`.
6. Аргументы командной строки для задачи 9: `n`, `m`, `p`, `kA`, `fA` (матрица `A`), `kB`, `fB` (вектор `B` размера `m×1`).
7. Аргументы командной строки для задачи 10: `n`, `m`, `k` (число столбцов `B`), `p`, `kA`, `fA`, `kB`, `fB` (матрица `B` размера `m×k`).
8. Ввод матрицы должен быть оформлен в виде подпрограммы, находящейся в отдельном файле.
9. Ввод матрицы из файла: `n × m` чисел построчно; программа выводит сообщение об ошибке при проблемах чтения.
10. Ввод матрицы и правой части по формуле $a_{i,j} = f(k, n, m, i, j)$:

    $$
    f(k,n,m,i,j) = \begin{cases}
      \max\{n,m\} - \max\{i,j\} + 1 & k = 1 \\
      \max\{i,j\} & k = 2 \\
      |i-j| & k = 3 \\
      \dfrac{1}{i+j-1} & k = 4
    \end{cases}
    $$

11. Решение должно быть оформлено в виде подпрограммы, находящейся в отдельном файле.
12. Программа должна содержать подпрограмму вывода на экран `n × m` матрицы, не более `p` строк и столбцов, формат элемента `" %10.3e"`.
13. Вывод результата в `main`: непосредственно результат (число или матрица), затем

    ```c
    printf ("%s : Task = %d Elapsed = %.2f\n",
            argv[0], task, t);
    ```

## Задачи

1. Функция: `n×n` матрица `A` симметрична ($a_{ij} = a_{ji}$)? Возвращает ненулевое значение, если да.
2. Функция, возвращающая след `n×n` матрицы `A`.
3. Подпрограмма, заменяющая `n×n` матрицу `A` на транспонированную.
4. Подпрограмма, заменяющая `n×n` матрицу `A` на $(A + A^t)/2$.
5. Подпрограмма, заменяющая `n×n` матрицу `A` на $(A - A^t)/2$.
6. Подпрограмма, переставляющая строки `i` и `j` местами в `n×m` матрице `A` (нумерация с 1).
7. Подпрограмма, переставляющая столбцы `i` и `j` местами в `n×m` матрице `A` (нумерация с 1).
8. Подпрограмма, прибавляющая к `j`-й строке матрицы `A` `i`-ю строку, умноженную на `γ` (нумерация с 1).
9. Подпрограмма, заменяющая вектор `c` (длины `n`) на произведение `A·b` (`A` — `n×m`, `b` — длины `m`).
10. Подпрограмма, заменяющая матрицу `C` (`n×k`) на произведение `A·B` (`A` — `n×m`, `B` — `m×k`).
