# Семестр 2 — Задание 12. Однонаправленные списки строк

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Задачи оцениваются независимо в двух группах: задачи 1–7 и задачи 8–10.
3. Аргументы командной строки для задач 1–10: `p` – максимальное количество выводимых значений списка, `filename` – имя файла со списком. Например, `./a.out 6 a.txt`.
4. В файле `filename` находится список строк, по одному элементу списка в строке (разделитель `\n` не включается в строку, длина строки не более 1233 байт). Концом списка считается конец файла.
5. Узел однонаправленного списка строк:

   ```c
   struct _node;                       /* предварительное объявление */
   typedef struct _node                /* объявление шаблона структуры */
     {
       char * string;                  /* данные - строка */
       struct _node * next;            /* использование объявления */
     } node;                           /* присвоение синонима шаблону */
   ```

6. Функции в задачах 1–7 имеют прототип `int function (node * head);`, в задачах 8–10 — `node * function (node * head);`.
7. Программа должна содержать функцию чтения списка, возвращающую −1, −2, −3 и т.д. при ошибке открытия файла/чтения/выделения памяти. Память под строку в узле выделяется по фактическому размеру строки (длина без `\n` плюс 1).
8. Программа должна содержать подпрограмму вывода на экран не более `p` первых элементов списка (после чтения, а в задачах 8–10 — и после применения решающей функции), каждый элемент на новой строке.
9. Количество выделений памяти в программе должно быть равно `2n`, где `n` – число строк во входном файле.
10. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
11. Вывод результата в `main`:
    - Для задач 1–7: `printf ("%s : Task = %d Result = %d Elapsed = %.2f\n", argv[0], task, res, t);`
    - Для задач 8–10: вывод нового состояния списка, затем `printf ("%s : Task = %d Length = %d Elapsed = %.2f\n", argv[0], task, len, t);`, где `len` — длина результирующего списка (вычисляется отдельной подпрограммой из `main`).

## Задачи

Функция получает указатель на начало однонаправленного списка строк:

1. Возвращает количество максимальных элементов списка.
2. Возвращает количество элементов списка, больших предыдущего элемента.
3. Возвращает количество нестрогих локальных максимумов списка.
4. Возвращает максимальное количество элементов между соседними нестрогими локальными максимумами.
5. Возвращает количество элементов в самом длинном участке строгого возрастания.
6. Возвращает количество участков постоянства списка.
7. Возвращает максимальное количество элементов между соседними участками постоянства.
8. Выбрасывает из списка все элементы, не меньшие следующего элемента. Возвращает указатель на начало результирующего списка.
9. Выбрасывает из списка все элементы, не меньшие своих двух соседей. Возвращает указатель на начало результирующего списка.
10. Выбрасывает из списка все участки нестрогого убывания. Возвращает указатель на начало результирующего списка.
