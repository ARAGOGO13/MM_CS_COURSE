# Семестр 2 — Задание 6. Матрицы: удаление строки/столбца по критерию

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки для задач 1–10: `m` – число строк, `n` – число столбцов, `p` – количество выводимых значений, `k` – номер формулы инициализации (0 = файл), `filename`. Например, `./a.out 6 4 5 0 a.txt` или `./a.out 2000 4000 6 1`.
3. Ввод матрицы должен быть оформлен в виде подпрограммы, находящейся в отдельном файле.
4. Ввод матрицы из файла: `m × n` чисел построчно; программа выводит сообщение об ошибке при проблемах чтения.
5. Ввод матрицы и правой части по формуле $a_{i,j} = f(k, m, n, i, j)$ — та же таблица, что в [Задании 4](../Tasks04/TASK.md) (с заменой `n,m` на `m,n`).
6. Решение должно быть оформлено в виде подпрограммы, находящейся в отдельном файле.
7. Программа должна содержать подпрограмму `print_matrix` вывода на экран `m × n` матрицы, не более `p` строк и столбцов, формат элемента `" %10.3e"`.
8. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
9. Во всех задачах требуется на месте исходной матрицы получить новую, проведя указанное преобразование; корректность формата проверяется вызовом `print_matrix`. Например, для задачи 1 (где на месте `m×n` получается `m×(n-1)`):

   ```c
   print_matrix (a, m, n, p);   /* печать исходной матрицы m x n */
   t = clock ();                /* время начала */
   res = solve1 (a, m, n);      /* подпрограмма решения задачи 1 */
   t = clock () - t;            /* время работы */
   print_matrix (a, m, n-1, p); /* печать новой матрицы m x (n-1) */
   ```

10. Вывод результата в `main`: непосредственно матрица (если изменялась), затем

    ```c
    printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
            argv[0], task, res, t);
    ```

## Задачи

Во всех задачах: функция получает `m × n` матрицу `A` вещественных чисел, `m`, `n`, формирует на месте `A` новую матрицу удалением строки/столбца по критерию и возвращает номер/индекс удалённых строки/столбца (нумерация с 1; при удалении и строки, и столбца — индекс $(i-1) \cdot n + (j-1)$).

1. Удалить столбец `j` с минимальным по модулю элементом $a_{ij}$ → матрица `m×(n-1)`.
2. Удалить строку `i` и столбец `j` с минимальным по модулю элементом $a_{ij}$ → матрица `(m-1)×(n-1)`.
3. Удалить столбец `j` с минимальной суммой модулей $\sum_{i=1}^{m} |a_{ij}|$ → `m×(n-1)`.
4. Удалить строку `i` и столбец `j` с минимальным значением $\sum_{k=1}^{n} |a_{ik}| + \sum_{k=1}^{m} |a_{kj}|$ → `(m-1)×(n-1)`.
5. Удалить столбец `j` с максимальным значением $\sum_{i=1, i\neq j}^{m} |a_{ij}|$ → `m×(n-1)`.
6. Удалить столбец `j` ($j = 1,\ldots,\min\{n,m\}$) с максимальным значением $\sum_{i=1, i\neq j}^{m} |a_{ij} - a_{jj}|$ → `m×(n-1)`.
7. Удалить столбец `j` с минимальным значением $\sum_{i=1}^{m} \left(a_{ij} - \dfrac{\sum_{k=1}^{m} a_{kj}}{m}\right)^2$ → `m×(n-1)`.
8. Удалить строку `i` и столбец `j` с максимальным значением $\sum_{k=1, k\neq j}^{n} |a_{ik}| + \sum_{k=1, k\neq i}^{m} |a_{kj}|$ → `(m-1)×(n-1)`.
9. Удалить строку `i` и столбец `j` с максимальным значением $\sum_{k=1, k\neq j}^{n} |a_{ik} - a_{ij}| + \sum_{k=1, k\neq i}^{m} |a_{kj} - a_{ij}|$ → `(m-1)×(n-1)`.
10. Удалить строку `i` и столбец `j` с минимальным значением $\sum_{k=1}^{n} \left(a_{ik} - \dfrac{\sum_{l=1}^{n} a_{il}}{n}\right)^2 + \sum_{k=1}^{m} \left(a_{kj} - \dfrac{\sum_{l=1}^{m} a_{lj}}{m}\right)^2$ → `(m-1)×(n-1)`.
