# Семестр 2 — Задание 5. Матрицы: норма, невязка, отклонение от единичной

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки для задач 1, 2, 7–11: `n`, `m` – размеры матрицы, `p` – количество выводимых значений, `k` – номер формулы инициализации (0 = файл), `filename`. Например, `./a.out 6 4 5 0 a.txt` или `./a.out 2000 4000 6 1`.
3. Аргументы командной строки для задач 3, 4: `n`, `m`, `p`, `kA`/`fA` (матрица `A`), `kb`/`fb` (вектор `b` размера `n×1`), `kx`/`fx` (вектор `x` размера `m×1`).
4. Аргументы командной строки для задач 5, 6: `n`, `m`, `p`, `kA`/`fA` (матрица `A`), `kB`/`fB` (матрица `B` размера `m×n`).
5. Ввод матрицы должен быть оформлен в виде подпрограммы, находящейся в отдельном файле.
6. Ввод матрицы из файла: `n × m` чисел построчно; программа выводит сообщение об ошибке при проблемах чтения.
7. Ввод матрицы и правой части по формуле $a_{i,j} = f(k, n, m, i, j)$ — та же таблица, что в [Задании 4](../Tasks04/TASK.md).
8. Решение должно быть оформлено в виде подпрограммы, находящейся в отдельном файле.
9. Программа должна содержать подпрограмму вывода на экран `n × m` матрицы, не более `p` строк и столбцов, формат элемента `" %10.3e"`.
10. Функция, реализующая задачу, не должна выделять или использовать дополнительную память.
11. Вывод результата в `main`: непосредственно матрица (если `A` изменялась), затем

    ```c
    printf ("%s : Task = %d Result = %e Elapsed = %.2f\n",
            argv[0], task, res, t);
    ```

## Задачи

1. Функция, возвращающая $\max_{i=1,\ldots,n} \sum_{j=1}^{m} |a_{ij}|$ для матрицы `A` размера `n×m`.
2. Функция, возвращающая $\max_{j=1,\ldots,m} \sum_{i=1}^{n} |a_{ij}|$.
3. Функция, возвращающая $\sum_{k=1}^{n} |r_k|$, где $r = Ax - b$ (`A` — `n×m`, `x` — длины `m`, `b` — длины `n`).
4. Функция, возвращающая $\max_{k=1,\ldots,n} |r_k|$, где $r = Ax - b$.
5. Функция, возвращающая $\max_{i=1,\ldots,n} \sum_{j=1}^{n} |r_{ij}|$, где $R = AB - I$ (`A` — `n×m`, `B` — `m×n`, `I` — единичная матрица).
6. Функция, возвращающая $\max_{j=1,\ldots,n} \sum_{i=1}^{n} |r_{ij}|$, где $R = AB - I$.
7. Функция, строящая на месте матрицы `A` матрицу `B` по формуле $b_{ij} = b_{i-1,j} + a_{i+1,j} + b_{i,j-1} + a_{i,j+1} - 4b_{ij}$ для $i=2,\ldots,n-1$, $j=2,\ldots,m-1$. Возвращает максимальный по модулю элемент `B`.
8. То же самое по формуле $b_{ij} = a_{i-1,j} + b_{i+1,j} + a_{i,j-1} + b_{i,j+1} - 4b_{ij}$.
9. То же самое по формуле $b_{ij} = b_{i-1,j} + a_{i+1,j} + a_{i,j-1} + b_{i,j+1} - 4b_{ij}$.
10. То же самое по формуле $b_{ij} = a_{i-1,j} + b_{i+1,j} + b_{i,j-1} + a_{i,j+1} - 4b_{ij}$.
11. Функция, переставляющая строки и столбцы `n×m` матрицы `A` так, чтобы на месте $a_{11}$ оказался элемент с максимальным модулем. Возвращает $|a_{11}|$ после перестановки.
