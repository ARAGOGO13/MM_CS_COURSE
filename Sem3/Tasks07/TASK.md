# Семестр 3 — Задание 7. Бинарное дерево объектов `student`: подсчёт характеристик

## Требования к программам

1. Программа работает с бинарным деревом объектов `student`, обёрнутым в класс `tree` (упорядоченное по `tree_node::operator<`, `student` — move-only):

   ```cpp
   enum class io_status { success, eof, format, memory };
   class student { /* ... как в Задании 3, move-only ... */ };

   class tree;
   class tree_node : public student
     {
     private:
       tree_node * left = nullptr;
       tree_node * right = nullptr;
     public:
       ...
       friend class tree;
     };
   class tree
     {
     private:
       tree_node * root = nullptr;
     public:
       ...
       void print (unsigned int r = 10, FILE *fp = stdout) const;
       io_status read (FILE * fp = stdin, unsigned int max_read = -1);
     private:
       static void delete_subtree (tree_node * curr);
       static void print_subtree (tree_node * curr, int level, int r, FILE *fp = stdout);
       static void add_node_subtree (tree_node * curr, tree_node *x);
     };
   ```

   Все функции задания — методы класса `tree`.

2. Программа должна получать все параметры в качестве аргументов командной строки: `r` – количество выводимых уровней дерева, `filename` – имя файла с деревом. Например, `./a.out 4 a.txt`.
3. Класс `tree` должен содержать функцию ввода дерева из файла: строки вида `Слово-i Целое-число-i`. Первый прочитанный объект `student` попадает в корень; каждый следующий добавляется в левое поддерево, если он меньше текущего узла (`tree_node::operator<`), и в правое иначе. Никакие другие функции, кроме чтения, не используют упорядоченность дерева. Концом ввода считается конец файла.
4. Решение задачи должно быть оформлено в виде подпрограммы в отдельном файле; дополнительную информацию извне получать запрещается.
5. Класс `tree` должен содержать подпрограмму вывода на экран не более `r` уровней дерева, с понятной структурой (отступы по уровням).
6. Программа должна выводить время, затраченное на решение.
7. Дерево не изменяется ни одной из функций, кроме последней задачи — поэтому все задачи вызываются из одной `main` (один исполняемый файл `a.out`), а вывод преобразованного дерева — сразу после вызова последней задачи.
8. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

## Задачи

Подпрограмма — член класса `tree`:

1. Возвращает количество концевых элементов дерева.
2. Возвращает длину наибольшей ветви дерева (в количестве элементов).
3. Возвращает максимальное количество элементов в одном уровне дерева.
4. Возвращает максимальную по модулю разность между глубинами левого и правого поддеревьев (в узлах дерева); глубина — количество уровней.
5. Возвращает количество элементов, имеющих ровно 1 потомка.
6. Удаляет все узлы (вместе со всеми потомками — поддерево целиком), имеющие наименьшее значение поля `value` среди всех узлов дерева. Возвращает количество удалённых узлов с этим значением (без учёта потомков).
