# Семестр 3 — Задание 9. Дерево с переменным числом ветвей (`student`)

## Требования к программам

1. Программа работает с деревом с переменным числом ветвей объектов `student` (представление "первый потомок / следующий сосед": каждый узел хранит указатель `down` на первого потомка и `level` на следующего "соседа" на своём уровне):

   ```cpp
   class student { /* ... move-only, cmp по name затем value ... */ };

   class tree;
   class tree_node : public student
     {
     private:
       tree_node * down = nullptr;  // первый потомок
       tree_node * level = nullptr; // следующий узел того же уровня (сосед)
     public:
       ...
       friend class tree;
     };
   class tree
     {
     private:
       tree_node * root = nullptr;
     public:
       ...
       void print (int r) const;
       int read (FILE * fp);
     private:
       static void delete_subtree (tree_node * curr);
       static void print_subtree (tree_node * curr, int level, int r);
       // Добавление узла x в список потомков curr с сохранением упорядоченности
       // по operator<: меньшие - в начало списка потомков, большие - в конец,
       // равные - в середину (перед первым не меньшим или после последнего не большего)
       static void add_node_subtree (tree_node* curr, tree_node* x);
     };
   ```

   Все функции задания — методы класса `tree`.

2. Программа должна получать все параметры в качестве аргументов командной строки: `r` – количество выводимых уровней, `filename`, `k` – параметр задачи. Например, `./a.out 4 a.txt 10`.
3. Класс `tree` должен содержать функцию ввода дерева из файла: строки вида `Слово-i Целое-число-i`. Первый объект попадает в корень; каждый следующий объект `x` добавляется в поддерево текущего узла как первый элемент списка потомков (если `x` меньше узла), последний элемент списка потомков (если больше), либо один из "средних" элементов (если равен — таких может быть несколько). Никакие другие функции, кроме чтения, не используют эту упорядоченность. Концом ввода считается конец файла.
4. Решение задачи должно быть оформлено в виде подпрограммы в отдельном файле; дополнительную информацию извне получать запрещается.
5. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d k = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, k, res, t);
   ```

6. Класс `tree` должен содержать подпрограмму вывода на экран не более `r` уровней дерева.
7. Программа должна выводить время, затраченное на решение.
8. Дерево не изменяется ни одной из функций, кроме последней задачи — поэтому все задачи вызываются из одной `main` (один исполняемый файл `a.out`), преобразованное дерево выводится сразу после вызова последней задачи.

## Задачи

Функция — член класса "дерево с переменным числом ветвей", получающая целое число `k`:

1. Возвращает количество элементов в узлах, имеющих ровно `k` потомков.
2. Возвращает количество элементов в поддеревьях, имеющих не более `k` вершин.
3. Возвращает количество элементов в поддеревьях, имеющих не более `k` уровней.
4. Возвращает количество элементов в поддеревьях, имеющих не более `k` узлов в любом уровне.
5. Возвращает количество элементов в `k`-м уровне дерева.
6. Возвращает количество элементов во всех ветвях длины не менее `k`, начиная с корня.
7. Удаляет все узлы (вместе со всеми потомками), имеющие значение поля `value` не превышающее `k`. Возвращает количество удалённых узлов с этим значением (без учёта потомков со значением `value > k`).
