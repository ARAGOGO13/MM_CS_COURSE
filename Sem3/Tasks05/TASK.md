# Семестр 3 — Задание 5. Двунаправленный список объектов `student` (класс `list2`)

## Требования к программам

1. Программа работает с двунаправленным списком объектов `student`, обёрнутым в класс `list2`; все функции задания — методы этого класса:

   ```cpp
   class student
     {
     private:
       char * name = nullptr;
       int    value = 0;
     public:
       ...
     };
   class list2_node : public student
     {
     private:
       list2_node * next = nullptr;
       list2_node * prev = nullptr;
     public:
       ...
       friend class list2;
     };
   class list2
     {
     private:
       list2_node * head = nullptr;
     public:
       ...
       int read (FILE *fp = stdin, unsigned int max_read = -1);
       void print (unsigned int r = 10, FILE *fp = stdout);
       unsigned int get_length ();
     };
   ```

2. Программа должна получать все параметры в качестве аргументов командной строки: `r` – количество выводимых значений, `filename` – имя файла со списком, `k` – параметр задачи. Например, `./a.out 4 a.txt 2`.
3. Класс `list2` должен содержать функцию `read` ввода списка из файла: строки вида `Слово-i Целое-число-i`; концом ввода считается конец файла или достижение `max_read` (по умолчанию — максимум для `int`).
4. Решение задачи должно быть оформлено в виде подпрограммы в отдельном файле; дополнительную информацию извне получать запрещается.
5. Класс `list2` должен содержать подпрограмму `print` вывода на экран списка длины не более `r`.
6. Класс `list2` должен содержать функцию `get_length` вычисления длины списка (для вывода длины исходного и результирующего списков).
7. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d Len Old = %u Len New = %u Elapsed = %.2f\n",
           argv[0], task, len1, len2, t);
   ```

   где `len1`/`len2` — длина исходного/результирующего списка.

## Задачи

Подпрограмма — член класса `list2`, получающая целое число `k`:

1. Циклический сдвиг элементов списка на `k` позиций вправо.
2. Выбросить все элементы $x_i$, для которых существует $j=1,\ldots,k$ такое, что $x_i > x_{i-j}$ (больше какого-либо из `k` предыдущих).
3. Выбросить все элементы $x_i$, для которых существует $j=1,\ldots,k$ такое, что $x_i > x_{i+j}$ (больше какого-либо из `k` следующих).
4. Выбросить все `k`-локальные максимумы — элемент $x_i$, для которого $x_{i-k} \le \cdots \le x_{i-1} \le x_i \ge x_{i+1} \ge \cdots \ge x_{i+k}$ и все $x_{i-j}$, $x_{i+j}$ существуют в последовательности.
5. Выбросить все участки постоянства длиной больше `k` целиком (наборы $x_i = x_{i+1} = \cdots = x_{i+j}$ при $j \ge k$).
6. Выбросить все участки нестрогого убывания длиной больше `k` целиком ($x_i \ge x_{i+1} \ge \cdots \ge x_{i+j}$ при $j \ge k$).
7. Выбросить целиком все участки между участками постоянства, имеющими длину больше `k`.
8. Выбросить целиком все участки между участками нестрогого возрастания, имеющими длину больше `k`.
