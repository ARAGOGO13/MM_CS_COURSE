# Семестр 3 — Задание 12. Красно-чёрное дерево (RB-дерево) объектов `student`

## Требования к программам

1. В программе реализованы: контейнер `student` (как в предыдущих заданиях: `read`, `print`, `operator<`) и **шаблонное красно-чёрное дерево**:

   ```cpp
   template <class T> class rb_tree;

   template <class T>
   class rb_tree_node : public T
     {
       enum class colors { invalid, red, black };
     private:
       rb_tree_node * left   = nullptr;
       rb_tree_node * right  = nullptr;
       rb_tree_node * parent = nullptr;
       colors color = colors::invalid;
     public:
       ...
       friend class rb_tree<T>;
     };

   template <class T>
   class rb_tree
     {
     private:
       rb_tree_node<T> * root = nullptr;
     public:
       rb_tree () = default;
       ...
       io_status read (FILE * fp = stdin) { ... }         // стандартная вставка с балансировкой RB-дерева
       void print (int r, FILE *fp = stdout) const { ... }
     };
   ```

   Все функции задания — методы класса `rb_tree<T>`, вызываются для `T = student`.

2. Программа должна получать все параметры в качестве аргументов командной строки: `r` – макс. количество выводимых уровней, `k` – параметр задачи, `filename`. Например, `./a.out 4 3 a.txt`.
3. Класс `rb_tree` должен содержать функцию ввода дерева из файла: строки вида `Слово-i Целое-число-i` (все записи различны). RB-дерево заполняется как упорядоченное дерево (стандартная вставка с перекрасками и вращениями для сохранения свойств красно-чёрного дерева). Никакие другие функции, кроме чтения, не используют упорядоченность.
4. Класс `rb_tree` должен содержать подпрограмму вывода на экран не более `r` уровней дерева, с понятной структурой.
5. Дерево не изменяется функциями задач — поэтому вся работа идёт в одной `main` (один исполняемый файл `a.out`):

   ```cpp
   int main (int argc, char *argv[])
   {
     // ... ввод аргументов ...
     rb_tree<student> *a = new rb_tree<student>;
     // чтение, вывод, вызов задач 1-5, вывод результатов и времени
     delete a;
     return 0;
   }
   ```

6. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d K = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, k, res, t);
   ```

## Задачи

Функция — член класса `rb_tree`, получающая целое число `k`:

1. Возвращает количество элементов `student` в поддеревьях, имеющих не более `k` вершин.
2. Возвращает количество элементов `student` в поддеревьях, имеющих не более `k` уровней.
3. Возвращает количество элементов `student` в поддеревьях, имеющих не более `k` узлов в любом уровне.
4. Возвращает количество элементов `student` в `k`-м уровне дерева.
5. Возвращает количество элементов `student` во всех ветвях длины `k`, начиная с корня.
