# Семестр 3 — Задание 3. Массив объектов `student` (move-only): фильтрация

## Требования к программам

1. Программа работает с массивом объектов класса `student`, на этот раз **move-only** (копирование запрещено — `student (const student&) = delete;`, `operator= (const student&) = delete;`, только перемещение):

   ```cpp
   enum class io_status { success, eof, format, memory };

   class student
   {
     private:
       char * name = nullptr;
       int    value = 0;
     public:
       student () = default;
       student (const student& x) = delete;
       student (student&& x);
       ~student ();
       student& operator= (const student& x) = delete;
       student& operator= (student&& x);
       void print () const;
       io_status read (FILE * fp);
       int operator> (const student& x) const;
       int operator< (const student& x) const;
       int operator== (const student& x) const;
     private:
       io_status init (const char * n, int v);
       void erase ();
       int cmp (const student& x) const;
   };
   ```

2. Программа должна получать все параметры в качестве аргументов командной строки.
3. Аргументы командной строки для задач 1–5: `n`, `p`, `s` (0 = файл), `filename`. Например, `./a04.out 4 4 0 a.txt` или `./a04.out 1000000 6 1`.
4. Аргументы командной строки для задач 6–10: доп. первый аргумент `x` — объект `student` (строка + целое число), затем `n`, `p`, `s`, `filename`.
5. Ввод массива должен быть оформлен в виде подпрограммы в отдельном файле; формат файла и формула инициализации — как в [Задании 2](../Tasks02/TASK.md).
6. Решение задачи должно быть оформлено в виде подпрограммы в отдельном файле, получающей массив и его длину (и доп. аргументы); дополнительную информацию извне получать и выводить что-либо на экран запрещается.
7. Программа должна содержать подпрограмму вывода на экран массива длины не более `p`.
8. Результатом работы каждой функции является измененный массив (выводится в `main`) и его новая длина (выводится в `main`).
9. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

## Задачи

Во всех задачах: функция получает массив `a[n]` объектов `student`, длину `n` (и объект `x`, где нужно), выбрасывает элементы согласно условию (не более $n + O(1)$ пересылок) и возвращает новую длину массива.

1. Выбросить элементы, меньшие предыдущего.
2. Выбросить элементы, большие 2-х предыдущих.
3. Выбросить элементы, меньшие соседних элементов.
4. Выбросить элементы, большие 2-х соседних элементов слева и справа.
5. Выбросить все участки убывания.
6. Выбросить все элементы, меньшие `x`.
7. Выбросить все подряд идущие элементы, меньшие `x`, оставляя только первый элемент в таком участке.
8. Выбросить все подряд идущие элементы, большие `x`, оставляя только первый и последний элементы в таких участках.
9. Выбросить все элементы, имеющие соседа (слева или справа), меньшего `x`.
10. Выбросить все участки возрастания, целиком состоящие из элементов, меньших `x`.
