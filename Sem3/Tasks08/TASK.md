# Семестр 3 — Задание 8. Бинарное дерево объектов `student`: структурные преобразования

## Требования к программам

1. Программа работает с тем же бинарным деревом `tree`/`tree_node` объектов `student`, что и в [Задании 7](../Tasks07/TASK.md), дополненным диагностическими методами:

   ```cpp
   int get_count_total () const; // количество элементов дерева
   int get_count_leaf () const;  // количество концевых элементов
   int get_count_1 () const;     // количество элементов с ровно 1 потомком
   int get_count_2 () const;     // количество элементов с ровно 2 потомками
   int get_height () const;      // длина наибольшей ветви
   int get_width () const;       // макс. количество элементов на одном уровне
   int get_balance () const;     // макс. |разность глубин левого/правого поддерева|
   void print (int r, FILE *fp) const;
   ```

2. Программа должна получать все параметры в качестве аргументов командной строки.
3. Аргументы командной строки для задач 1–4: `r` – количество выводимых уровней, `filename`. Например, `./a.out 4 a.txt`.
4. Аргументы командной строки для задач 5–7: `r`, `filename`, `name`, `value` — задают объект `student {name, value}`. Например, `./a.out 4 a.txt Student 5`.
5. Задачи оцениваются независимо в двух группах: задачи 1–4 и задачи 5–7.
6. Класс `tree` должен содержать функцию ввода дерева из файла (формат и правило упорядочивания — как в [Задании 7](../Tasks07/TASK.md)).
7. Решение задач должно быть оформлено в виде подпрограмм в отдельном файле; дополнительную информацию извне получать запрещается.
8. Программа должна выводить время, затраченное на решение.
9. Вывод результата в `main` (после каждого преобразования — контрольные величины `T,L,C1,C2,H,W,B` до и после):

   ```cpp
   tree a;
   // ... Initialize the tree ...
   printf ("Original tree:\n");
   a.print (r);
   int T=a.get_count_total(), L=a.get_count_leaf(), C1=a.get_count_1(), C2=a.get_count_2();
   int H=a.get_height(), W=a.get_width(), B=a.get_balance();
   printf ("%s : Task = %d T = %d L = %d C1 = %d C2 = %d H = %d W = %d B = %d\n",
           argv[0], task, T, L, C1, C2, H, W, B);
   double t = clock ();
   solve (); // решение
   t = (clock() - t)/CLOCKS_PER_SEC;
   printf ("Modified tree:\n");
   a.print (r);
   // T,L,C1,C2,H,W,B пересчитываются заново
   printf ("%s : Task = %d T = %d L = %d C1 = %d C2 = %d H = %d W = %d B = %d Elapsed = %.2f\n",
           argv[0], task, T, L, C1, C2, H, W, B, t);
   ```

## Задачи

Подпрограмма — член класса `tree`:

1. Преобразовать дерево так, чтобы в каждой вершине стоял максимальный элемент в её поддереве.
2. Преобразовать дерево так, чтобы в каждом уровне элементы шли в порядке возрастания `value`, методом "нахождения минимума".
3. Правостороннее вращение по правилу: для узла `A` без левого потомка, у которого есть неконцевой правый потомок `B`, найти в поддереве `B` "самый левый" концевой элемент `C` (путь от `C` к `B` идёт только через левых потомков); отсоединить `C`, поставить на место `A` (родитель `C` = родитель `A`, правый потомок `C` = `B`, левый потомок `C` = `A`), `A` становится концевым с родителем `C`.
4. Зеркальное преобразование к задаче 3: для узла `A` без правого потомка с неконцевым левым потомком `B`, найти "самый правый" концевой элемент `C` в поддереве `B` и переставить аналогично.
5. Удалить все поддеревья глубиной не менее `k` (аргумент `value` объекта `x`, `name` = строка `s`), у которых в поле `name` каждого узла входит строка `s`.
6. Удалить все поддеревья, имеющие ветвь длиной не менее `k`, такую что в поле `name` каждого узла этой ветви входит строка `s`.
7. Удалить все поддеревья, начинающиеся с уровня из не менее `k` подряд идущих элементов уровня, в каждый из которых в поле `name` входит строка `s`.
