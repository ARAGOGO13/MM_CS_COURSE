# Семестр 3 — Задание 4. Однонаправленный список объектов `student`

## Требования к программам

1. Программа работает с однонаправленным списком объектов `student`:

   ```cpp
   class student
     {
     private:
       char * name = nullptr;
       int    value = 0;
     public:
       ...
     };
   class list_node : public student
     {
     private:
       list_node * next = nullptr;
     public:
       ...
     };
   ```

   Сам объект "список" в этом задании не используется, все функции получают указатель на начало списка.

2. Программа должна получать все параметры в качестве аргументов командной строки: `r` – количество выводимых значений в списке, `filename` – имя файла со списком. Например, `./a.out 4 a.txt`.
3. Ввод списка должен быть оформлен в виде подпрограммы в отдельном файле.
4. Ввод списка из файла: строки вида `Слово-i Целое-число-i` (слово — алфавитно-цифровая последовательность без пробелов). Концом ввода считается конец файла.
5. Решение задачи должно быть оформлено в виде подпрограммы в отдельном файле, получающей указатель на начало списка; дополнительную информацию извне получать и выводить что-либо на экран запрещается.
6. Программа должна содержать подпрограмму вывода на экран списка длины не более `r`, по одному элементу на строку.
7. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

## Задачи

Функция получает указатель на начало однонаправленного списка объектов `student`:

1. Возвращает количество максимальных элементов списка.
2. Возвращает количество элементов списка, больших предыдущего элемента.
3. Возвращает количество локальных максимумов списка (элемент, не меньший своих соседей, если они есть).
4. Возвращает 1, если список не убывает, 2 — если не возрастает, 3 — если постоянен, 4 — если недостаточно данных для принятия решения, 0 — если не монотонен.
5. Возвращает длину максимального участка строгого возрастания списка.
6. Возвращает количество участков постоянства списка.
7. Возвращает максимальное расстояние между соседними участками постоянства (в количестве элементов между ними).
8. Возвращает длину максимального участка нестрогой монотонности списка.
