# Работа на ЭВМ и программирование

Домашние задания за 4 семестра курса «Работа на ЭВМ и программирование»: от битовых операций и работы с указателями в C до структур данных, шаблонов и STL в C++, вплоть до собственной мини-СУБД с клиент-серверным взаимодействием по сети.

## Стек

- **Семестры 1–2** — язык C (99/11), сборка через `gcc` + `Makefile`.
- **Семестры 3–4** — язык C++ (14), сборка через `g++` + `Makefile`.

## Структура репозитория

Каждый семестр — отдельная папка `SemN`, внутри — папки `TasksNN` по числу домашних заданий. В каждой `TasksNN`:

- `TASK.md` — условие задания (изначально было в `TasksNN.pdf`, переведено в Markdown для читаемости и поиска по репозиторию),
- исходники решений (`.c`/`.cpp`/`.h`),
- `Makefile` для сборки.

## Семестры

| Семестр | Язык | Темы |

| [Семестр 1](Sem1/README.md) | C | Битовые операции, работа с файлами, массивы, сортировки, строковые функции |

| [Семестр 3](Sem3/README.md) | C++ | ООП-контейнеры, связные списки, бинарные деревья, шаблоны, B-деревья, красно-чёрные деревья |

Подробное описание и ссылки на условия каждого задания — в `README.md` соответствующего семестра.
