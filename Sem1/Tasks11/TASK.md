# Семестр 1 — Задание 11. Построчная обработка текстовых файлов

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки для задач 1–2:
   1. `fin` – имя входного файла,
   2. `fout` – имя выходного файла,
   3. `s` – строка.

   Например, `./a01.out a.txt b.txt "abcd"`.

3. Аргументы командной строки для задач 3–7: `fin`, `fout`, `s` – строка, `t` – строка. Например, `./a03.out a.txt b.txt "abcd" "(){}[] ;+-*/="`.
4. Аргументы командной строки для задачи 8: `fin`, `fout`, `s`, `l` – строка, `r` – строка. Например, `./a08.out a.txt b.txt "abcd" "(<" ">)"`.
5. Результатом работы каждой функции является измененный файл (не выводится в `main`) и возвращаемое значение (выводится в `main`).
6. Вывод результата работы функции в функции `main`:

   ```c
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

   где `res` – результат работы функции, `t` – время работы. Вывод должен производиться в точности в таком формате, чтобы можно было автоматизировать обработку запуска многих тестов.

## Задачи

Во всех задачах функция получает имя `a` входного текстового файла неизвестной длины, имя `b` выходного файла и строковые параметры, и возвращает количество затронутых строк или −1, −2 и т.д. при ошибке чтения/открытия файла.

1. Вывести в файл `b` те строки файла `a`, в которые входит строка `s` (прописные и соответствующие строчные латинские буквы считаются совпадающими).
2. Вывести в файл `b` те строки файла `a`, у которых конец строки совпадает со строкой `s` (например, при `s="ab"` — строки, завершающиеся `"ab"`).
3. Вывести в файл `b` строки файла `a`, заменяя каждое вхождение строки `s` на строку `t`.
4. Вывести в файл `b` те строки файла `a`, которые не совпадают со строкой `s`, не учитывая пробельные символы (пробельным считается символ, содержащийся в строке `t`).
5. Вывести в файл `b` те строки файла `a`, которые состоят только из символов строки `s`, не учитывая пробельные символы (из `t`).
6. Вывести в файл `b` те строки файла `a`, в которые входит строка `s`, не учитывая пробельные символы (из `t`).
7. Вывести в файл `b` те строки файла `a`, в которые входит строка `s`, начиная с первого непробельного символа (пробельные — из `t`).
8. Вывести в файл `b` строки файла `a`, заменяя каждое вхождение строки `s` на строку, образованную строками `l`, `s`, `r` (например, при `s="a"`, `l="("`, `r=")"` строка `"abc"` заменяется на `"(a)bc"`).
