# Семестр 4 — Задание 2. Поиск по словам с деревьями-индексами (B-дерево, RB-дерево, AVL-дерево)

## Требования к программам

1. Задачи оцениваются независимо в двух группах: задачи 1–5 и задачи 6–9.
2. Программа должна получать все параметры в качестве аргументов командной строки.
3. Аргументы командной строки для задачи 1: `fin`, `fout`, `s`, `t`, `m` – параметр (основание) B-дерева поиска. Например, `./a01.out a.txt b.txt "abcd" "(){}[] ;+-*/=" 2`.
4. Аргументы командной строки для задач 2, 6–9: `fin`, `fout`, `s`, `t`. Например, `./a01.out a.txt b.txt "abcd" "(){}[] ;+-*/="`.
5. Аргументы командной строки для задач 3–5: `fin`, `fout`, `s`, `t`, `x`. Например, `./a03.out a.txt b.txt "abcd" "(){}[] ;+-*/=" "ABCD"`.
6. Результатом работы каждой функции является измененный файл (не выводится в `main`) и возвращаемое значение (выводится в `main`).
7. Вывод результата работы функции в функции `main`:

   ```cpp
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

## Задачи

Во всех задачах: функция получает имя `a` входного текстового файла неизвестной длины, имя `b` выходного файла, строки `s`/`t` (и `x`, где нужно), указатель на счётчик `r`; словом называется последовательность символов без пробельных символов (пробельные — символы из строки `t`). Функция строит индекс по словам строки `s` **до** обработки текстового файла (для быстрого поиска), затем разбирает каждую строку файла `a` на слова и ищет их в индексе. Возвращает `io_status`.

1. Вывести строки, имеющие общее слово со строкой `s`. Слова строки `s` размещаются **в B-дереве строк по основанию `m`** (строки и массивы в узлах — как `std::unique_ptr<T>`).
2. Вывести строки, состоящие только из слов строки `s`. Слова строки `s` размещаются **в красно-чёрном дереве строк** (`std::unique_ptr<char[]>`).
3. Вывести строки файла `a`, заменяя каждое вхождение слова из `s` на соответствующее по номеру слово из `x` (недостающие слова — пустые строки, лишние — игнорируются). Пары (слово `s`, слово `x`) размещаются **в AVL-дереве пар строк** (ключ поиска — слово из `s`).
4. Вывести строки, где есть слово, удовлетворяющее условию из `x` по отношению к слову из `s` (условия: `<`, `>`, `<=`, `>=`, `=`, `<>`; число условий = число слов; условие "ИЛИ" между словами). Например, для `s="ab cd"`, `x="= >"` подходят слова, либо равные `"ab"`, либо лексикографически больше `"cd"`.
5. То же самое, но условие "И" между словами. Например, для `s="ab cd"`, `x="> <"` подходят слова, лексикографически больше `"ab"` **и** меньше `"cd"`.
6. Вывести строки со словом, совпадающим с образцом `s`, где `_` — один любой символ (эскейпинг `\_`, `\\`).
7. Вывести строки со словом, совпадающим с образцом `s`, где `%` — 0 или более любых символов (варианты `text%`, `%text`, `%text%`, `text1%text2`; эскейпинг `\%`, `\\`).
8. Вывести строки со словом, совпадающим с образцом `s`, где `[n-m]` — один символ с кодом в диапазоне `n...m` (эскейпинг `\[`, `\]`, `\\`; непарные скобки — ошибка).
9. То же самое, но `[bn-m]` — один символ с кодом **вне** диапазона `n...m` (эскейпинг `\[`, `\]`, `\b`, `\\`).
