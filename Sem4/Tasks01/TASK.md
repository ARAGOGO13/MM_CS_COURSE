# Семестр 4 — Задание 1. AVL-дерево объектов `student`: подсчёт по подстроке

## Требования к программам

1. В программе должны быть реализованы следующие структуры данных:
   - Контейнер данных `student` (move-only, `unique_ptr<char[]>` для имени, лексикографическое сравнение по `name`, затем по `value`; `io_status` с кодами `success, eof, format, memory, open, create`).
   - Шаблонное **AVL-дерево** `avl_tree<T>` / `avl_tree_node<T>` (узел хранит `left`, `right`, целочисленный `balance`):

     ```cpp
     template <class T> class avl_tree;
     template <class T>
     class avl_tree_node : public T
       {
       private:
         avl_tree_node * left  = nullptr;
         avl_tree_node * right = nullptr;
         int balance = 0;
       public:
         avl_tree_node () = default;
         ...
         friend class avl_tree<T>;
       };
     template <class T>
     class avl_tree
       {
       private:
         avl_tree_node<T> * root = nullptr;
       public:
         avl_tree () = default;
         ...
         int read (FILE * fp = stdin);
         void print (int r, FILE *fp = stdout);
       };
     ```

2. Достаточно реализовать только функцию добавления элементов в дерево с сохранением его сбалансированной структуры (стандартные повороты AVL-дерева при вставке). Эта функция вызывается при чтении элементов. Дерево удаляется целиком в деструкторе, поэтому удаление отдельного узла с сохранением структуры реализовывать не нужно.
3. Все функции задания — методы класса `avl_tree<student>`.
4. Программа должна получать все параметры в качестве аргументов командной строки: `r` – макс. количество выводимых уровней, `s` – строка (параметр задачи), `filename`. Например, `./a.out 4 "abc" a.txt`.
5. Класс `avl_tree` должен содержать функцию ввода дерева из файла: строки вида `Слово-i Целое-число-i` (все записи различны). Никакие другие функции, кроме чтения, не используют упорядоченность дерева.
6. Класс `avl_tree` должен содержать подпрограмму вывода на экран не более `r` уровней дерева, с понятной структурой.
7. Дерево не изменяется функциями задач — вся работа идёт в одной `main` (один исполняемый файл `a.out`):

   ```cpp
   int main (int argc, char *argv[])
   {
     // ... ввод аргументов ...
     avl_tree<student> *a = new avl_tree<student>;
     // чтение, вывод, вызов задач 1-5, вывод результатов и времени
     delete a;
     return 0;
   }
   ```

8. Каждая задача вызывается дважды: сначала для константной строки `s_all`, содержащей все буквы латинского алфавита (строчные и прописные), затем для строки `s`, заданной аргументом командной строки.
9. Вывод результата в `main`:

   ```cpp
   const char * s_all = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ";
   printf ("%s : Task = %d S = %s Result = %d Elapsed = %.2f\n",
           argv[0], task, s_all, res_all, t_all);
   printf ("%s : Task = %d S = %s Result = %d Elapsed = %.2f\n",
           argv[0], task, s, res, t);
   ```

## Задачи

Функция — член класса `avl_tree`, получающая строку `s` и учитывающая только узлы, у которых поле `name` состоит **только** из символов строки `s`:

1. Возвращает количество концевых элементов дерева, удовлетворяющих условию.
2. Возвращает максимальное количество элементов в одной ветви дерева, удовлетворяющих условию.
3. Возвращает количество поддеревьев дерева, удовлетворяющих условию.
4. Возвращает максимальное количество элементов в одном уровне дерева, удовлетворяющих условию.
5. Возвращает максимальную по модулю разность между количеством элементов, удовлетворяющих условию, в левом и правом поддереве (по всем узлам дерева).
