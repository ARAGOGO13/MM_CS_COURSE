# Семестр 4 — Задание 10. STL-контейнеры: дедупликация и подсчёт строк (vector/list + sort)

## Требования к программам

1. Программа должна получать все параметры в качестве аргументов командной строки.
2. Аргументы командной строки для задач 1–4: `fin` – имя входного файла, `fout` – имя выходного файла. Например, `./a01.out a.txt b.txt`.
3. Аргументы командной строки для задач 5–6: `fa` – имя входного файла `a`, `fb` – имя входного файла `b`, `fout` – имя выходного файла, `t` – строка-разделитель. Например, `./a01.out a.txt b.txt c.txt "(){}[] ;+-*/="`.
4. Результатом работы каждой функции является изменённый файл (не выводится в `main`) и возвращаемое значение (выводится в `main`).
5. Вывод результата в `main`:

   ```cpp
   printf ("%s : Task = %d Result = %d Elapsed = %.2f\n",
           argv[0], task, res, t);
   ```

## Задачи

1. Функция получает имя `a` текстового файла неизвестной длины, имя `b` выходного файла и ссылку на `int r`; выводит в `b` строки файла `a`, не повторяющиеся в файле, сохраняя исходный порядок (повторяющаяся строка выводится один раз, на месте первого появления). Например, для `"b","a","c","b","a","d"` ответ — `"b","a","c","d"`. `r` получает количество выведенных строк; функция возвращает `io_status`.
   Реализация: контейнер `std::vector<std::pair<std::string,int>>`, куда считываются строки с их номерами; сортировка по первому полю (`std::sort`) и удаление повторов (оставляя минимальный номер), затем повторная сортировка по номеру и вывод в исходном порядке.
2. То же самое, что и задача 1, но на основе контейнера `std::list<std::pair<std::string,int>>` (та же логика: `std::sort` по первому полю, дедупликация, повторная сортировка по номеру).
3. Функция получает `a`, `b`, `r`; выводит в `b` строки файла `a` в формате `"номер число_повторений строка"`. Например, для `"b","a","c","b","a","d"` ответ — `"1 2 b", "2 2 a", "3 1 c", "4 2 b", "5 2 a", "6 1 d"`. `r` получает количество **уникальных** строк (в примере — 4).
   Реализация: `std::vector<std::pair<std::string, std::pair<int,int>>>` (строка, (номер, счётчик)); сортировка по строке — подсчёт повторов; сортировка обратно по номеру — вывод.
4. То же самое, что и задача 3, но на основе `std::list<std::pair<std::string, std::pair<int,int>>>`.
5. Функция получает имена `a`, `b`, имя `c` для вывода, строку-разделитель `t`, ссылку на `int r`; выводит в `c` строки файла `b` в формате `"число_общих_слов_со_строками_a строка_b"`. Словом называется последовательность без символов `t`. Например, для `t=" "`, `a = {"b","a","c","b","a","d"}`, `b = {"b a","a b","c","e"}` ответ в `c` — `{"2 b a","2 a b","1 c","0 e"}`; `r` получает суммарное число общих слов (в примере — 5).
   Реализация: `std::vector<std::string>` со словами файла `a`, отсортированный `std::sort` и очищенный от повторов `std::unique`; поиск слов из `b` — через `std::binary_search`.
6. То же самое, что и задача 5, но на основе `std::list<std::string>` со словами `a`, отсортированного `std::sort` и очищенного `std::unique`; поиск слов из `b` — через `std::find` (линейный).
