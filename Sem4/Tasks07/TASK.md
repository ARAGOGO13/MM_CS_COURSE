# Семестр 4 — Задание 7. Мини-СУБД: две структуры быстрого поиска (v5)

Развитие [Задания 6](../Tasks06/TASK.md): вместо одной структуры быстрого поиска реализуются **две** одновременно (например, по имени и по телефону).

## Требования к программам

Те же, что в [Задании 6](../Tasks06/TASK.md) (класс "База данных", список + структуры быстрого поиска, команды `insert`/`select`/`delete`/`quit`, `config.txt` для параметров `k`/`m`).

## Задачи

Требуется реализовать **две** из 38 структур для быстрого поиска, перечисленных в [Задании 6](../Tasks06/TASK.md).

## Вариант

Файл `group_209-tasks07.pdf` содержал список группы с номерами двух вариантов (столбцы `name`/`phone`) для каждого студента и был заменён на [`my-variant.md`](my-variant.md) — публикация чужих ФИО нежелательна.
